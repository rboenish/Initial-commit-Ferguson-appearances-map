    11→    """

    13→    Each new number = current number + previous number.
    14→
    15→    Args:
    16→        iterations: Number of iterations to run
    17→
    18→    Returns:

    28→        next_val = current + prev
    29→        sequence.append(next_val)
    30→        prev = current
    31→        current = next_val
    32→
    33→    return sequence
